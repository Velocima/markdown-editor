# Markdown Editor

A simple markdown editor

## Installation & Usage

### Installation

```sh
git clone git@github.com:Velocima/markdown-editor.git
cd markdown-editor
npm install
```

**NB: Requires NodeJs and NPM**

### Usage

**Available scipts:**

Run the dev server at [http://localhost:3000/](http://localhost:3000/)

```sh
npm run start
```

---

Build the app to the `build` folder

```sh
npm run build
```

---

Run all test suites

```sh
npm test
```
